# A7 Documentation

A7 is a small ahead-of-time compiler. It lowers `.a7` source to a single Zig
0.16 source file, then the host Zig toolchain builds a native binary. The
compiler is Python. There is no A7 runtime.

## What changed in this rebuild

- The docs site is now static HTML generated by Bun.
- The previous client-rendered stack is gone.
- Public docs are refactored into a smaller corpus with stable raw markdown.
- Agent entry points remain first-class: `llms.txt`, `llms-full.txt`, and
  `/docs/<slug>.md`.

- `docs/STATUS.md` for gaps and priorities.
- `docs/RELEASE.md` for release gates.
